Data Scholarship (NAA1661) Final Project: Topic Pitches

艺术、媒体与量化美学 (The Quantitative Aesthetics of Art and Media)

优势与背景: 这个方向的数据集（如 Spotify, IMDB）极其干净且丰富。用统计学去解构艺术能产生极强的反差感和戏剧张力，非常容易引起观众的共鸣。同时，它完美契合各种极具视觉冲击力的高级图表。

四个可选分支:

  1. 音乐（Teammate A）: 爵士乐与摇摆舞曲的非线性声学动力学。提取 Danceability, Acousticness 等声学特征，用极坐标柱状图展示复杂流派的演化。
  2. 文本 (Teammate B): 经典电影剧本的情感拓扑。清洗台词文本，利用 Pyecharts 制作钻石形状的词云图来呈现核心意象与情感波动。
  3. 地理 (Teammate C): 艺术流派的地理迁徙。利用大都会艺术博物馆等藏品数据，使用 Geo 地理图表追踪百年间全球艺术重镇的转移。
  4. 商业 (Teammate D): 口碑与票房的博弈机制。分析预算、流派与最终评分的分布，利用直方图和散点图寻找颠覆常理的 Outliers。

Presentation Storytelling (整体串联): "The Anatomy of Art" —— 我们的展示可以从一段 5 秒的音乐或一句经典的电影台词开场。首先从最微观的个体表达切入（剧本情感与音乐声学特征），带领观众看看感性艺术背后的绝对理性；随后镜头拉远，在宏观地理画布上展现这些艺术流派是如何跨越大洲传播的；最后落地现实，探讨在商业法则下，这些艺术作品是如何在票房和口碑中博弈的。

社交计算与数字行为动力学 (Social Computing & Digital Behavior)

优势与背景: 代入感无敌。我们每天都在经历这些数字交互，且各类论坛、社交媒体的数据非常容易在 Kaggle 找到。这是展示人机交互和平台机制设计的绝佳沙盒。

四个可选分支:

  1. 机制 (Teammate A): 问答社区的算法激励与用户生命周期。分析点赞、徽章系统如何非线性地影响用户的发帖积极性与长期留存率。
  2. 网络 (Teammate B): 信息茧房与回音室效应。构建网络图谱，看看特定话题下，不同观点是如何在各自的圈层中形成拓扑结构的。
  3. 趋势 (Teammate C): 网络热梗的衰变曲线。使用主题河流图展示不同网络流行语或话题在一段时间内的搜索热度起伏与生命周期。
  4. 商业 (Teammate D): 众筹平台的信任机制。探究 Kickstarter 上的筹款目标、更新频率等变量是如何决定项目最终成败的。

Presentation Storytelling (整体串联): "The Algorithm of Us" —— 开场抛出一个大家每天都会刷手机的共鸣现象。第一部分揭秘平台底层的激励机制是如何设计出来“留住”我们的；第二部分展示我们在这种机制下，是如何被困在信息茧房里互动的；第三部分拉长视角，看这些互动产生的话题浪潮是如何起伏消亡的；最后，我们用众筹的案例证明，数字世界里的这些点击和注意力，最终是如何转化为真金白银的信任。

跨文化交流与全球学习网络 (Cross-Cultural Communication & EdTech)

优势与背景: 高度契合 UNNC 的国际化背景。这个切入点非常 Human-centric，我们很容易结合自身的跨文化交流经验，讲出有温度、有共情力的数据故事。

四个可选分支:

  1. 协作 (Teammate A): 在线学习社群的协作机制设计。分析 Study Buddy 或学习小组的组队模式如何影响全球用户的最终完课率。
  2. 行为 (Teammate B): 语言学习平台的用户粘性。探究游戏化学习（Gamification）对不同文化背景用户的激励差异。
  3. 聚类 (Teammate C): 全球教育资源的分布图谱。结合宏观经济，用 3D 散点图将不同国家的教育投入、识字率进行多维聚类。
  4. 文本 (Teammate D): 跨文化视角的应用评价。利用分词与关键字提取，对比同款软件在不同国家应用商店里的评价痛点。

Presentation Storytelling (整体串联): "Bridging the Global Gap" —— 开场先用 3D 聚类图展现全球教育资源分配的宏观不均衡；接着，视角缩小到具体的在线学习平台，看看不同文化背景的用户在使用这些工具时的痛点（评价分析）和学习习惯（游戏化激励差异）；最后，提出解决方案，用数据证明优秀的“在线协作与社群机制”能够跨越地理屏障，提升最终的学习效果。

科技生态与创新网络 (Tech Ecosystems & Developer Communities)

优势与背景: 充满极客精神，数据非常干净结构化（如 GitHub API 或开发者调查）。能展现我们对前沿科技、创投以及硬核编程领域的专业素养。

四个可选分支:

  1. 创投 (Teammate A): 湾区科技创投网络的多变量聚类分析。展示 AI、Web3 等前沿领域的资金流向和技术迭代周期。
  2. 协作 (Teammate B): GitHub 项目的开源生命周期。分析 Commit 频率、Issue 响应时间等协作模式如何造就常青的开源项目。
  3. 地理 (Teammate C): 黑客松（Hackathon）与技术社区的全球版图。将赛事的举办地、参与人数进行地理空间映射。
  4. 趋势 (Teammate D): 编程语言和技术栈的世代交替。用动态图表展示过去十年间 Python 等工具的流行度演变。

Presentation Storytelling (整体串联): "The Blueprint of Innovation" —— 这是一个关于“创新是如何诞生”的故事。我们先追踪资本的流向（湾区创投网络），看看钱在哪；然后描绘草根创新的发源地（全球 Hackathon 版图），看看人在哪；接着深入代码层面，剖析这些极客们是如何在 GitHub 上远程协作改变世界的；最后，我们用一条时间线，预测未来他们手中的技术栈将如何更新换代。

城市微出行与集体人类动力学 (Urban Mobility & Collective Dynamics)

优势与背景: 逻辑严密，非常适合展现我们这几周学到的 df.drop_duplicates() 等复杂数据清洗技巧。虽然主题比较常见，但能做出极其酷炫的地理热力图。

四个可选分支:

  1. 交通 (Teammate A): 微出行的时空非线性需求激增。结合天气与早晚高峰，应用 Mapbox 密度热力图呈现共享单车的拥堵潮汐现象。
  2. 环境 (Teammate B): 城市空气质量与人类活动的耦合。分析 PM2.5 数据，用设置了特定 xbins 尺寸的直方图展示危险天气的分布。
  3. 基建 (Teammate C): 交通事故的时空高发热点。清洗市政记录，找出事故高发地段与道路设计的相关性。
  4. 经济 (Teammate D): 居住空间的演变轨迹。结合房价与商圈分布，描绘城市发展的扩张轨迹。

Presentation Storytelling (整体串联): "The City as a Living Organism" —— 把城市比作一个庞大的生命体。首先，我们用直方图描绘它的“呼吸环境”（空气质量与季节的关系）；随后，切入它的“骨架与阵痛”（交通事故热点与基建）；接着，看它的“肌肉生长”（经济与居住空间的扩张）；最后，用一张动态的 Mapbox 热力图，展示在这一切之上，血液（人群微出行）是如何在城市经脉中产生非线性脉动的。
